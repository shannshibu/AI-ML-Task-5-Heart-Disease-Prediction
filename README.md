# Heart Disease Prediction

## Overview

This project predicts whether a patient has heart disease using supervised machine learning.

## Dataset
Heart Disease Dataset

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Models

- Logistic Regression
- Decision Tree

## Results

Logistic Regression performed better with an accuracy of 83.15%.
